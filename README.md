Shop bán hàng điện tử - Electroshop

Một website bán hàng điện tử cơ bản 

Yêu cầu Python 3.10 trở lên, Node.js >= v17

-- Cách chạy dự án --
B1: Git clone - https://github.com/Sken2504/Electroshop
B2: Tải các dependencies cần thiết - npm install
B3: Chạy dự án - node src/server.js
